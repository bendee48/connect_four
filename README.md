# The Odin Project 

## Ruby - RSpec - Connect Four

**A Ruby project exploring TDD with RSpec, by building the game connect four.**

*The Game*
  - 2 players.
  - Choose a column 1 - 7 to place your piece.
  - 4 in a row wins.
